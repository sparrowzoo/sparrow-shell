后端代码示例

    public ViewWithModel hello() throws BusinessException {
    //hello 默认映射到/template/hello.jsp
    return ViewWithModel.forward("hello", new HelloVO("我来自遥远的sparrow 星球,累死我了..."));
    }

前端获取参数

hello 为HelloVO的类名,自动将POJO 结尾字符（DTO,VO,BO）去除

hi \[(${hello.hello})]
